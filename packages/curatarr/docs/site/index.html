CURATARR

Intelligent media curation for Jellyfin.

Curatarr combines scanning, metadata sync, disambiguation, verify, and scout workflows in one modern tool.

See Capabilities Get Started API Docs

Scan

Run ffprobe-backed library scans and track file-level quality coverage.

Verify

Catch metadata mismatches and reduce false upgrades before they happen.

Scout

Use Prowlarr-integrated candidate discovery with configurable scoring rules.

Quick start

Install dependencies, build, and start Curatarr on your local machine.

npm install
cp config/config.example.yaml config/config.yaml
npm run build
npm run serve
