# Curatarr API Docs Source

Source files for Curatarr OpenAPI documentation.

- OpenAPI spec source: `openapi.yaml`
- Swagger UI source page: `index.html`

Build to deployable static output:

```bash
npm run docs:api:build
```

Output directory:

- `../site/docs/api/`

Intended hosted URL:

- `https://curatarr.video/docs/api/`
